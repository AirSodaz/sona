# Sona 用户指南

[English](user-guide.md) | [简体中文](user-guide.zh-CN.md) | [项目 README](../README.zh-CN.md)

这份指南面向桌面端用户，帮助您完成 Sona 的安装、首次配置、本地转录、文本整理与导出。

## 1. Sona 适合做什么

Sona 是一款以隐私优先为目标的转录编辑器，适合希望默认在本机完成语音转文字流程，而不是先把音频上传到云端的用户。

如果您有以下需求，Sona 会比较适合：

- 用 `实时录音` 记录会议、课堂、访谈或语音笔记
- 用 `批量转录` 处理已有的音频或视频文件
- 在按时间轴组织的编辑器中逐段检查和修改文本
- 在自行配置服务后，使用 `LLM 润色` 或 `翻译`
- 将结果导出为常见字幕或纯文本格式
- 按需启用 `语音输入法`，把识别结果直接输入到其他应用

如果您主要是为了 `实时字幕` 而来，请直接查看 `实时录音` 章节；如果您主要想用 `语音输入法`，请直接查看 `历史记录与设置` 中关于 `设置 > 快捷键` 的说明。

## 2. 安装与启动

对大多数用户来说，最简单的方式是直接安装最新发布版本：

- 从 [GitHub Releases](https://github.com/AirSodaz/sona/releases/latest) 下载 Sona。
- 启动应用。
- 如果您需要从源码构建，请参考[项目 README](../README.zh-CN.md)。

Sona 首次打开时，可能会直接进入 `首次运行设置`。如果您暂时跳过，后续应用会继续通过提醒横幅提示您完成离线初始化。

### 需要 CLI？

如果您需要命令行批量转录，请查看专门的 [CLI 指南](cli.zh-CN.md)。本指南只覆盖桌面端日常使用流程。

## 3. 首次运行设置

在本地离线转录可用之前，Sona 需要先完成离线模型初始化。

### 前置条件

- Sona 可以正常启动。
- 如果您准备直接下载推荐模型包，需要联网。

### 操作步骤

1. 启动 Sona，等待 `首次运行设置` 出现。
2. 阅读欢迎步骤。Sona 推荐的首条成功路径是 `麦克风 -> 实时录音`。
3. 点击 `继续`。
4. 在模型步骤点击 `下载推荐模型`。如果所需模型已经存在，界面会直接显示 `继续`。
5. 等待推荐离线模型下载并解压完成。
6. 进入麦克风步骤并允许麦克风访问权限。
7. 如果权限被拒绝，请在处理系统权限提示后点击 `重新请求权限`。
8. 选择您希望 `实时录音` 默认使用的麦克风。
9. 点击 `前往实时录音`。

### 完成后会发生什么

- Sona 会应用推荐的本地离线转录配置。
- 应用会切换到 `实时录音`。
- 如果设置仍未完成，提醒横幅可以再次打开引导。

### 说明

- 首次引导的目标是尽快让本地离线转录进入可用状态。
- 您可以在引导过程中点击 `稍后`，之后再通过提醒横幅返回。
- 如果您主动关闭提醒横幅，它将不再在首页显示，直到设置完成。
- 后续可以在 `设置 > 模型中心` 中改用其他模型。
- 默认麦克风可以在 `设置 > 输入设备` 中重新调整。

## 4. 实时录音

如果您希望边录音边生成转录内容，请使用 `实时录音`。

### 前置条件

- 您已经完成 `首次运行设置`，或已在 `设置 > 模型中心` 中配置 `实时录音模型`。
- 如果要从麦克风录音，系统已经授予麦克风权限。

### 操作步骤

1. 点击 `实时录音` 页签。
2. 在开始录音前，从下拉框选择输入源：
   `麦克风` 或 `桌面音频`。
3. 点击 `开始录音`。
4. 通过波形和计时器观察采集状态。
5. 需要临时中断时使用 `暂停`，要结束本次录音时使用 `停止`。
6. 点击 `参数设置`，按需调整 `字幕模式` 或 `语言`。
7. 如果需要悬浮字幕窗口，请打开 `实时字幕`。
8. 如需调整字幕浮窗的置顶、点击穿透、尺寸、宽度、颜色或启动行为，请前往 `设置 > 字幕设置`。

### `实时字幕` 是做什么的

- `实时字幕` 是 `实时录音` 页面里的 `悬浮字幕（系统音频）` 开关，适合您主要想把系统音频以悬浮字幕方式显示出来的场景。
- 它可以单独开启，不需要先开始录音；如果您之后再开始 `实时录音`，两者也可以并行使用。
- `设置 > 字幕设置` 负责控制字幕窗口的启动行为、置顶、点击穿透、字体大小、宽度、颜色和背景透明度。

### 完成后会发生什么

- 新的转录分段会实时出现在右侧编辑器中。
- 当前活跃分段会跟随录音状态变化。
- 停止录音后，当前转录内容仍可继续编辑、润色、翻译、导出并保存到历史记录。

### 说明

- 默认情况下，`Ctrl + Space` 用于开始或停止实时录音。
- 录音过程中按 `Space` 可以暂停或继续。
- `参数设置` 主要控制 `字幕模式` 和 `语言`，不是完整的 LLM 润色入口。
- 如果提示缺少模型，请重新完成引导，或到 `设置 > 模型中心` 手动配置。

## 5. 批量转录

如果您已经有音频或视频文件，并希望在后台排队转录，请使用 `批量转录`。

### 前置条件

- 您已经在 `设置 > 模型中心` 中配置 `批量转录模型`。
- 输入文件属于支持的音频或视频格式。

### 操作步骤

1. 点击 `批量转录` 页签。
2. 将文件拖入导入区域，或者点击 `选择文件`。
3. 添加一个或多个文件到队列。
4. 观察队列侧栏和当前处理项状态。
5. 如需继续追加文件，点击 `添加更多文件`。
6. 点击 `参数设置`，按需调整 `字幕模式` 或 `语言`。
7. 文件处理完成后，在主编辑器中检查结果。

### 完成后会发生什么

- Sona 会按 `待处理`、`处理中`、`完成`、`失败` 等状态管理队列。
- 已完成的文件会载入主编辑器，供您继续编辑、翻译和导出。

### 说明

- 如果未配置离线批量模型，Sona 会重新打开引导，而不是直接开始转录。
- `设置 > 模型设置` 中的 `VAD Buffer Size` 与 `Max Concurrent Transcriptions` 会影响批处理体验。

## 6. 转录编辑与播放

当 Sona 生成转录分段之后，右侧编辑器就是主要工作区。

### 前置条件

- 您已经通过 `实时录音`、`批量转录` 或 `历史记录` 载入了转录内容。

### 操作步骤

1. 在编辑器中查看分段列表。
2. 点击时间戳可以跳转到对应播放位置。
3. 双击分段文本，或使用编辑按钮进入编辑状态。
4. 按 `Enter` 保存当前分段。
5. 按 `Shift + Enter` 在编辑时插入换行。
6. 使用合并按钮将当前分段与下一段合并。
7. 使用删除按钮在确认后删除分段。
8. 按 `Ctrl + F` 搜索当前转录内容。
9. 如果当前条目带有音频，使用底部播放器进行播放、暂停、跳转、倍速和音量控制。

### 完成后会发生什么

- 您可以逐段修正文本。
- 时间轴与播放位置会保持联动。

### 说明

- 只有在编辑某一段时，编辑工具栏才会显示。
- 工具栏支持 `撤销`、`重做`、加粗、斜体、下划线和换行。
- 搜索结果可以在不同分段之间快速跳转。

## 7. LLM 润色、翻译与摘要

Sona 的 LLM 功能是可选能力。离线转录本身不依赖它，但 `LLM 润色`、`翻译` 与 `AI 摘要` 需要先在 `设置 > LLM 服务` 中完成配置。

### 前置条件

- 您已经有转录分段。
- 您已经在 `设置 > LLM 服务` 中配置了要使用的功能。

### LLM 服务设置步骤

1. 打开 `设置 > LLM 服务`。
2. 在 `功能模型绑定` 中分别为 `润色模型`、`翻译模型` 和 `摘要模型` 指定模型。
3. 在 `Provider 凭据` 中展开要使用的 provider，填写相应的连接信息，例如 `Base URL`、`API Key`、`Endpoint`、`Deployment Name` 或 provider 特有字段。
4. 点击 `测试连接` 验证配置。
5. 确认所需功能已经绑定模型后，再返回主界面使用。

### `LLM 润色` 操作步骤

1. 确认 `设置 > LLM 服务` 中已经指定 `润色模型`。
2. 点击 `LLM 润色` 按钮。
3. 按需选择：
   `LLM 润色`、`重新转录`、`撤销`、`重做`、`高级设置`。
4. 如果需要管理 `Auto-Polish`、`Auto-Polish Frequency`、`关键词`、`场景预设`、`自定义上下文`，请打开 `高级设置`。

### `翻译` 操作步骤

1. 确认 `设置 > LLM 服务` 中已经指定 `翻译模型`。
2. 点击 `翻译` 按钮。
3. 选择目标语言。
4. 点击 `开始翻译` 或 `重新翻译`。
5. 使用 `显示双语` 或 `隐藏双语` 控制编辑器中的双语显示。

### `AI 摘要` 操作步骤

1. 确认 `设置 > LLM 服务` 中已经指定 `摘要模型`。
2. 打开任意一个已经有分段内容的转录。
3. 在编辑器顶部的摘要面板中切换 `通用`、`会议` 或 `课堂` 模板。
4. 点击 `生成` 创建当前模板的摘要，之后也可以点击 `重新生成` 手动刷新。
5. 如果需要把摘要用于其他地方，点击 `复制`。
6. 如果后续又编辑、润色或重新转录了原始文本，旧摘要会保留，但会显示已过期提示，直到您手动重新生成。

### 完成后会发生什么

- `LLM 润色` 会直接更新转录文本。
- `翻译` 会把翻译结果保存到每个分段，并可显示在原文下方。
- `AI 摘要` 会把只读摘要作为附属结果保存在转录旁边，不会改写原文。

### 说明

- `润色模型`、`翻译模型` 与 `摘要模型` 是分开配置的。您可以让它们共用同一个 provider，也可以分别设置。
- 翻译可以使用 `Google Translate (Free)`、`Google Translate (API)` 这类专用翻译 provider；`LLM 润色` 则需要真正的 LLM provider 与模型。
- `AI 摘要` 也需要真正的 LLM provider 与模型，不支持 `Google Translate` 系列 provider。
- 当前支持的翻译目标语言包括 `简体中文`、`English`、`Japanese`、`Korean`、`French`、`German`、`Spanish`。
- 只有当当前转录来自已保存的历史条目时，菜单中才会出现 `重新转录`。
- 当前摘要结果为只读文本，不会并入导出文件；如需复用，请直接从摘要面板复制。

## 8. 导出转录

只要编辑器里至少有一个分段，就可以导出。

### 前置条件

- 当前编辑器中已经有转录内容。

### 操作步骤

1. 点击顶部栏中的 `导出转录` 按钮。
2. 在 `导出转录` 弹窗中填写 `文件名`。
3. 选择 `导出目录`。
4. 选择输出格式：
   `SubRip (.srt)`、`WebVTT (.vtt)`、`JSON (.json)`、`Plain Text (.txt)`。
5. 选择导出模式：
   `原文`、`译文`、`双语`。
6. 点击 `导出`。

### 完成后会发生什么

- Sona 会把转录内容写入您选择的路径和格式。
- 如果已有翻译内容，您可以导出纯译文或双语版本。

### 说明

- 只有在当前转录至少包含一个翻译分段时，`译文` 和 `双语` 才可用。
- `原文` 模式始终可用。

## 9. 历史记录与设置

`历史记录` 用于重新打开之前的工作，`设置` 用于管理默认行为。

### `历史记录` 使用步骤

1. 点击 `历史记录` 页签。
2. 按标题或转录内容搜索。
3. 使用 `全部类型`、`录音`、`批量导入` 等条件按类型筛选。
4. 使用 `任意时间`、`今天`、`最近 7 天`、`最近 30 天` 等条件按时间筛选。
5. 点击某个条目即可重新载入。
6. 如果要批量删除，先进入选择模式。

### 建议优先了解的设置区域

- `设置 > 常规`
  主题、界面语言、字体、托盘行为、更新检查
- `设置 > 输入设备`
  麦克风选择、系统音频设备选择、麦克风增强、录音时静音
- `设置 > 字幕设置`
  实时字幕启动行为、点击穿透锁定、置顶、字体大小、宽度、颜色、背景透明度
- `设置 > 模型中心`
  `实时录音模型`、`批量转录模型`，以及可下载的识别模型、标点模型、VAD 模型
- `设置 > 模型设置`
  `转录设置`、`ITN`、`VAD Buffer Size`、`Max Concurrent Transcriptions`、`恢复默认设置`
- `设置 > 词汇`
  `文本替换` 规则集与 `热词` 规则集
- `设置 > LLM 服务`
  功能模型绑定与 Provider 凭据
- `设置 > 快捷键`
  实时录音快捷键与 `语音输入法`
- `设置 > 关于`
  源码、日志与更新相关入口

### `语音输入法`

- `语音输入法` 适合在聊天窗口、文档编辑器、表单等其他应用中直接输入文字。
- 前往 `设置 > 快捷键`，启用 `语音输入法`，设置全局快捷键，并选择 `按住说话` 或 `切换模式`。
- `按住说话` 更适合短句输入；`切换模式` 更适合连续口述，按一次开始，再按一次停止。
- `语音输入法` 依赖同一套离线实时转录能力，因此也需要先配置可用的 `实时录音模型`。

### 说明

- 在 `设置 > 词汇` 中，`热词` 按每行一个的方式填写，也支持 `关键词 :2.0` 这类带权重的写法；目前这项能力对 Transducer 与 Qwen3 ASR 模型尤其相关。
- 在 `设置 > 词汇` 中，`文本替换` 适合用来统一术语或修正重复出现的识别结果。

## 10. 常见问题与排障

### Sona 一直提示我完成首次设置

- 打开提醒横幅，把模型和麦克风步骤完成。
- 如果您之前跳过了引导，请确认 `设置 > 模型中心` 中已经配置好 `实时录音模型` 和 `批量转录模型`。
- 如果您曾主动关闭提醒横幅，请在 Sona 再次提示时从相关入口重新进入引导。

### `实时录音` 无法开始

- 检查系统麦克风权限。
- 确认已经配置 `实时录音模型`。
- 确认当前输入源是否正确：`麦克风` 或 `桌面音频`。

### `批量转录` 无法开始

- 确认已经配置 `批量转录模型`。
- 确认文件扩展名属于支持格式。
- 如果 Sona 提示格式不支持，请先转换文件格式再重试。

### `LLM 润色` 或 `翻译` 不可用，或者执行失败

- 确认 `设置 > LLM 服务` 中已经填写正确的 provider 凭据。
- 确认功能本身已经绑定模型：润色需要 `润色模型`，翻译需要 `翻译模型`。
- 先使用 `测试连接` 验证连通性。
- 如果您使用自定义接口地址或本地服务，例如 `Ollama`，请先确认那个服务本身可访问。

### 找不到 `Auto-Polish`

- 请打开 `LLM 润色 > 高级设置`，在那里可以配置 `Auto-Polish`、频率、关键词、场景预设和自定义上下文。

### 导出时只有 `原文`

- 只有当当前转录已经包含翻译文本时，`译文` 和 `双语` 才会出现。

### `实时字幕` 没有显示出来

- 请先回到 `实时录音` 页面，确认已经打开 `实时字幕`；`设置 > 字幕设置` 只负责窗口行为和外观，不负责启动入口。
- 如果您只是想显示系统音频字幕，不必先开始录音，直接开启 `实时字幕` 即可。
- `实时字幕` 依赖同一套离线实时转录能力，因此也需要先配置可用的 `实时录音模型`。

### `语音输入法` 无法工作

- 先在 `设置 > 快捷键` 中启用 `语音输入法`。
- 确认快捷键设置符合您的预期。
- 语音输入法依赖同一套离线实时转录能力，因此也需要先配置可用的实时转录模型。

### 看不到播放控件

- 只有当当前转录关联了音频来源时，底部播放器才会显示，例如已保存的录音条目或已处理完成的文件。

### 我想从源码构建或参与开发

- 请查看[项目 README](../README.zh-CN.md)中的源码构建与开发命令说明。
